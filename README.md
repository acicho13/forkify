## Run Locally

Clone the project

```bash
  git clone https://github.com/amandacichosz/forkify.git
```

Go to the project directory

```bash
  cd forkify
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Deployment

To deploy this project run

```bash
  npm run build
```
